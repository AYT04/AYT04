The 10-Stage Installation Process

The platform coordinates a structural loop to educate deployment technicians manually on low-level Linux actions rather than abstracting procedures away:

1. Verification Testing

Validates network capabilities via ping www.ayt04.xyz and tests firmware via efivar -l.

2. Partition Table Zapping

Loads gdisk expert operations flags (x then z) to zero out structural sectors cleanly.

3. cgdisk Space Allocation

Slices drive geometries cleanly into individual sectors matching required parameters.

4. Filesystem Format Structuring

Styles FAT32 format maps to boot space targets and standard EXT4 journaling to directory clusters.

5. Folder Mounting Hierarchies

Binds root partition 3 to primary live mount destination /mnt, linking boot & home partitions safely inside branches.

6. Pacstrap Core Deployments

Downloads base utilities, operating system kernels, and device hardware firmwares directly from ranked localized mirrors.

7. genfstab Immutable Mapping & Chroot

Writes block tables using immutable PARTUUID tags before executing arch-chroot /mnt to move inside the system drive filesystem.

8. Regional Settings Calibration

Generates localization configuration parameters, binds UTC motherboard internal clocks, and turns on SSD blocks cleaning optimizations.

9. Bootloader Installation

Deploys modern lightweight systemd-boot modules directly to hardware partition headers via bootctl install.

10. Display Greeters Setup

Enables graphical window server frameworks (Xorg) and registers sddm.service to prompt graphical user dashboards automatically upon system reboots.
