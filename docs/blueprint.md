# **App Name**: TubeAlert

## Core Features:

- Channel Configuration: Allow users to input and store YouTube Channel IDs and API Key via a simple form.
- Background Check: Check for new video uploads from configured channels using the YouTube API.
- Notification Display: Display new video notifications in a clear list.

## Style Guidelines:

- Primary color: White or light grey for a clean background.
- Secondary color: Dark grey for text and important elements.
- Accent: Blue (#3498db) for interactive elements and highlights.
- Simple, single-column layout for easy navigation.
- Use recognizable icons for channel management and settings.
