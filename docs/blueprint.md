# **App Name**: Project Showcase

## Core Features:

- Project Display: Display project details including title, description, and links.
- Project Navigation: Implement a clear and intuitive navigation menu to browse projects.
- Project Search: Add a search bar to quickly find specific projects.
- GitHub Comments: Allow users to comment on projects using their GitHub accounts.
- Google Comments: Allow users to comment on projects using their Google accounts.

## Style Guidelines:

- Primary color: Dark blue (#24292F) for a professional look.
- Secondary color: Light gray (#F6F8FA) for backgrounds and accents.
- Accent: Teal (#26A69A) for interactive elements.
- Clean and modern sans-serif fonts for readability.
- Use a grid-based layout for a structured and organized appearance.
- Subtle transitions and animations for a polished user experience.
